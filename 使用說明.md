# TourPage AI 文案生成器 - 使用說明

## 🚀 現在您可以直接在網頁上設定 API Key！

不需要編輯任何檔案，一切都在瀏覽器中完成。

## 📝 使用步驟

### 步驟 1：開啟應用程式

訪問 http://localhost:3000

### 步驟 2：設定 API Key

第一次使用時，您會看到一個藍色的設定卡片，顯示：

```
┌─────────────────────────────────────────┐
│ 🔑 設定 Google Gemini API Key          │
├─────────────────────────────────────────┤
│ 請前往 Google AI Studio 取得您的免費   │
│ API Key，然後貼上到下方欄位中。        │
│                                         │
│ [輸入欄位] [儲存 API Key 按鈕]         │
│                                         │
│ 您的 API Key 會安全地儲存在瀏覽器...   │
└─────────────────────────────────────────┘
```

### 步驟 3：取得 API Key

1. 點擊卡片中的「Google AI Studio」連結
2. 使用 Google 帳號登入
3. 點擊「Create API Key」
4. 選擇或建立一個專案
5. 複製產生的 API Key

### 步驟 4：儲存 API Key

1. 返回應用程式
2. 將 API Key 貼上到輸入欄位
3. 點擊「儲存 API Key」按鈕
4. 完成！

### 步驟 5：開始使用

設定完成後，您會看到綠色的確認訊息：

```
┌─────────────────────────────────────────┐
│ ✓ API Key 已設定 (AIzaSyB...)          │
│                          [變更 API Key] │
└─────────────────────────────────────────┘
```

現在您可以：
1. 上傳 CSV 檔案
2. 設定行銷參數
3. 生成 AI 文案
4. 下載結果

## 🔒 安全性

- API Key 只儲存在**您的瀏覽器**中
- 不會傳送到任何第三方伺服器
- 只用於直接呼叫 Google Gemini API
- 可以隨時變更或清除

## 💡 常見問題

**Q: 如果換了瀏覽器還需要重新設定嗎？**
A: 是的，因為 API Key 儲存在瀏覽器的 localStorage 中。

**Q: 清除瀏覽器資料會怎樣？**
A: API Key 會被清除，需要重新設定。

**Q: API Key 會過期嗎？**
A: 除非您在 Google AI Studio 中刪除或重新產生，否則不會過期。

**Q: 如何變更 API Key？**
A: 點擊綠色訊息中的「變更 API Key」按鈕即可。
